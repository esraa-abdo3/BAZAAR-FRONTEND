# 🛍️ Bazaary

Bazaary is a modern digital platform that transforms traditional bazaars into an organized online experience. It enables organizers to manage bazaars, brands, products, and visitors through dedicated dashboards while providing customers with a seamless shopping experience.

---

## 🚀 Live Demo

https://bazaar-frontend-ten.vercel.app/

---

## 📌 Features

### 👤 Customer

- Browse active bazaars.
- Search for brands and products.
- View detailed product information.
- Add products to Wishlist.
- Add products to Cart.
- Secure checkout process.
- Responsive design.
- AI Assistant for helping visitors discover brands and products.

---

### 🏪 Brand Dashboard

- Brand registration.
- Manage brand profile.
- Add products.
- Edit products.
- Delete products.
- Product image uploads.
- Track products inside bazaars.

---

### 🎪 Bazaar Dashboard

- Manage bazaar information.
- Manage participating brands.
- View registered brands.
- Edit bazaar profile.
- Upload bazaar images.
- Package management.

---

### 🛡️ Admin Dashboard

The admin has full control over the entire platform.

#### Dashboard

- Platform statistics.
- Analytics.
- Reports.

#### Bazaar Management

- View all bazaars.
- Add new bazaar.
- Edit bazaar.
- View bazaar details.

#### Brand Management

- View all brands.
- Add new brand.
- Edit brand.
- Block/Delete brand.

#### Product Management

- View all products.
- Add new product.
- Edit product.
- Block/Delete product.
- Upload multiple product images.

#### Users Management

- View users.
- Delete users.

#### Orders

- View all orders.
- View order details.

#### Settings

- Update platform settings.
- Change platform logo.
- Update social media links.

---

## 🤖 AI Assistant

The platform includes an AI Assistant that helps users:

- Find products quickly.
- Recommend brands.
- Answer visitor questions.
- Improve user experience.
